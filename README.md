# C# Vector WinForms Application

## Description

This is a C# Windows Forms application created as an academic project.  
The project demonstrates object-oriented programming, working with vector data structures, file operations, streams, cloning, and JSON serialization.

The application provides a graphical user interface for creating vectors, choosing a vector implementation type, performing vector operations, and saving/loading data.

## Features

- Graphical interface built with Windows Forms
- Work with `ArrayVector`
- Work with `LinkedListVector`
- Vector creation using a table
- Vector display
- Vector norm calculation
- Vector cloning
- Changing cloned vector data
- Byte file output
- Byte file input
- Text file output
- Text file input
- JSON save
- JSON load

## Technologies

- C#
- .NET
- Windows Forms
- Visual Studio
- Object-Oriented Programming
- File Streams
- JSON Serialization

## Project Structure

- `Form1.cs` — main form logic
- `Form1.Designer.cs` — Windows Forms UI layout
- `Form1.resx` — form resources
- `Program.cs` — application entry point
- `Models/` — vector classes and application models
- `Properties/` — project properties and settings
